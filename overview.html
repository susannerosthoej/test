1 Overview

This e-book contains an overview of SDA-related material: practical information as well as an overview of the folders on the H-drive (KBF-SDA).

Der kommer kap til hver folder …

The e-book is generated in the folder named overview on the H-drive from the file overview.Rmd. It is accessed from the short cut overview.html in the root of the H-drive, but is physically stored in the overview folder.

To update the overview:

  • install the bookdown package
  • each chapter should be stored in a separate Rmd file
    • first line in the Rmd files should include the title of the chapter (starting the line with a single #)
    • if you generate a new chapter you need to add a chunk to the main file overview.Rmd, specifying the name of the new Rmd-file as child in a chunk
  • knit the file named overview.Rmd (don’t knit the Rmd files corresponding to the chapters)
  • Voila!
